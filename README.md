Стек:

Python 3.9

Django 3.1

SQLite3


Критические баги:

1.Любой пользователь может удалять любые записи

2.При создании поста в блоге можно выбрать любого автора

3.При создании поста в блоге можно выбрать любое время публикации


Проблемы:

1.Дата поста реализована в формате DD:MM:YY, а не DD:MM:YY HH:MM

2.Нет возможности перейти в блог автора через пост


Не реализовано:

1.Система подписок на авторов

2.Пометка поста прочитанным

3.Рассылка уведомлений авторам

4.Использование Docker


Запуск приложения:

1.Перейти в терминале в директорию приложения

2.Использовать в терминале py manage.py runserver

3.В браузере перейти на 127.0.0.1:8000
